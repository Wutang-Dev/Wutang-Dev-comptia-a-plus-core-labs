# MacBook Lab Rebuild

## Objective
Prepare a MacBook Pro as a portable IT lab for studying CompTIA A+ Core 2.

## Steps Completed

1. Reinstalled macOS using Internet Recovery
2. Installed VirtualBox
3. Created Windows 11 VM
4. Downloaded Ubuntu ISO
5. Installed Git
6. Created GitHub repository for Core 2 labs
7. Cloned repository to local machine

## Lab Environment

Host OS: macOS  
Virtualization: VirtualBox  

Virtual Machines:
- Windows 11 Enterprise Evaluation
- Ubuntu 24.04 (planned)

## Purpose

This environment will be used to practice:

- Windows administration
- Linux troubleshooting
- virtualization
- Git documentation
- IT support scenarios
