# Wutang-Dev CompTIA A+ Core Labs

Practical hands-on labs used to reinforce concepts while studying for **CompTIA A+ Core 2 (220-1102)**.

This repository documents real troubleshooting exercises performed in a home lab environment.

---

## Lab Environment

Host Machine:
- MacBook Pro (Intel)
- macOS

Virtualization:
- VirtualBox

Virtual Machines:
- Windows 11 Enterprise Evaluation
- Ubuntu 24.04

---

## Lab Structure

| Lab | Description |
|----|----|
| MacBook Lab Rebuild | Reinstalled macOS and built a portable IT lab environment |
| Windows Labs | Windows administration and troubleshooting |
| Linux Labs | Linux command line and system troubleshooting |
